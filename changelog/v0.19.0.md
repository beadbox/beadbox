## What's New in v0.19.0

### Added

- **bd version catchup formula**: Repeatable process for catching up to new bd CLI versions.

### Changed

- **Faster real-time updates**: DOLT_HASHOF_TABLE polling replaces row-scanning queries (2x faster server-mode change detection).
- **Faster workspace scanner**: Removed obsolete DerivePort range scan (was 10s, now instant).
- **bd 0.61.0 minimum**: Version gate ensures compatibility with UUID primary keys and ephemeral ports.
- **Simplified workspace dialog**: Local Path is now primary, server connection moved to Advanced section.

### Fixed

- **Comment deletion on bd 0.61+**: UUID primary keys now handled correctly.
- **Comment change detection**: Replaced fragile MAX(id) with COUNT(*) for UUID compatibility.
- **Error messages show actual port**: Replaced hardcoded 3307 with real port from dolt-server.port.
- **Missing port file no longer floods logs**: Single warning instead of repeated ENOENT errors.
- **Removed dead SERVER badge**: Server-only workspaces no longer show confusing badge.
